What's new in v2.1 beta

Version 2.1 of World Historical Gazetteer extends the existing Collection feature. Previously a WHG Collection could consist only of 2 or more entire public datasets. Users can now create collections not only from entire datasets but from individual place records discovered in the Search and Browse screens. We differentiate these now as Dataset Collections (the initial option) and Place Collections, which are new.

Also, a number of "under the hood" improvements have been made to improve reliability of the system, and to improve communication and workflows for data contributors.

Place Collections

Place Collections are authored works—sets of places that are related in some way according to the collection's author/creator. Our initial use case is for teaching in historical subjects at secondary- and college-levels. Students might be assigned to create a Place Collection as a product of research on a given topic or one of their choosing within a broad theme.

  • a set of place records from any public contributed dataset, selected from the WHG index and/or database
  • optional annotations on each place record, describing briefly why it is part of the collection
  • an image that illustrates the collection topic
  • an essay, uploaded as a PDF file
  • up to three links to supporting external web resources
  • descriptive keywords

This guide to creating and publishing Place Collections explains the workflow in detail.
